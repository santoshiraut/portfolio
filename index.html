Discover India

This project is a tourism website showcasing popular Indian destinations. The homepage features a "Get Started" button that leads to a list of favorite places: Taj Mahal, Golden Temple, Mysore Palace, and Varanasi Temple. Each destination has a brief description and an image. Clicking on a place takes users to a detailed view with a carousel of images and more information. The site uses Bootstrap for layout and custom CSS for styling, with JavaScript enabling smooth navigation between sections.
